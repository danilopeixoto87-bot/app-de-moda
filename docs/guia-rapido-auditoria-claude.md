# Guia Rápido - Auditoria Claude Code

## 1) Executar auditoria
Abra o Claude Code no projeto e use o conteúdo de:
- docs/comando-unico-auditoria-claude.md

## 2) Resultado esperado
O Claude deve gerar:
- docs/auditoria-pre-sprints-resultado.json

## 3) Critério de aceite da auditoria
- JSON válido
- decisão GO ou NO_GO
- findings com evidência por arquivo
- backlog 80/20 priorizado

## 4) Próximo passo após auditoria
Enviar o arquivo `docs/auditoria-pre-sprints-resultado.json` para análise e transformação em plano de sprint detalhado.
